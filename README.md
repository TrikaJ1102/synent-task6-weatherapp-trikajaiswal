# Modern Weather App 🌦️

## Objective

A modern GUI-based Weather Application built using Python and Tkinter that fetches real-time weather data using the OpenWeather API.

---

## Features

✅ Search weather by city  
✅ Real-time weather updates  
✅ Temperature display  
✅ Humidity information  
✅ Wind speed  
✅ Weather condition & description  
✅ Dynamic weather icons  
✅ Error handling  
✅ Modern dark-themed UI  
✅ Keyboard ENTER support

---

## Technologies Used

- Python
- Tkinter
- Requests
- Pillow
- OpenWeather API

```bash

```

### 3. Run Application

```bash
python main.py
```
